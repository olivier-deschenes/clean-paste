# Clean Paste

Clean Paste is a Chrome extension that removes formatting when you paste text.

It works like paste-as-plain-text, but automatically.

You can turn it on or off for each website by clicking the extension icon.

## How It Works

- Click the extension icon on any website.
- Green means Clean Paste is enabled for that site.
- Red means Clean Paste is disabled for that site.
- When enabled, pasted text is inserted without style formatting.
